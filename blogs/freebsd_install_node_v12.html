FreeBSD 11.1安装NodeJS v12失败

安装NodeJS:

这里用懒惰的办法，直接安装二进制：pkg install node（ports源码安装实在太慢了）

安装过程是没有任何问题的，但是我想 node -v看下版本测试是否真的安装成功的时候。

然后就出事了：Undefined symbol "nghttp2_submit_origin" nodejs FreeBSD

我搜了许久都没有与我情况类似的，哎！小众的劣势凸显。最终我还是发挥了FreeBSD的优势：官方的文档异常详细， 最后找到了如下图所示

node12依赖的包

其实真正导致这个问题的是缺少 libnghttp2但是我看到自带python 版本是python27-2.7.13，为了不出其他的幺蛾子，索性把依赖的库都更新到最新了。

把依赖的包依次安装或更新，问题瞬间解决。完美！
