安装Nginx web服务器

我这里使用的服务器操作系统是FreeBSD 11.1

FreeBSD 系统有两种安装软件的方式:第一种是port进行源码安装,这种安装比较慢.所以我采用第二种安装方式,pkg安装二进制包

    pkg install nginx

在FreeBSD上,安装完后需要记住以下几个目录地址:

Nginx 网站文档目录: /usr/local/www/nginx

Nginx 配置文件地址: /usr/local/etc/nginx/nginx.conf

Nginx 软件地址: /usr/local/sbin/nginx

Nginx服务器常用的基本命令:

Nginx 服务器启动: /usr/local/sbin/nginx -s start

Nginx 服务器关闭: /usr/local/sbin/nginx -s stop

Nginx 服务器重载配置文件: /usr/local/sbin/nginx -s reload

如果需要在Nginx在系统重启后自动开启,需要配置:/etc/rc.conf

    ...
    nginx_enable="YES"
    ...
